# 📌 Flask Task Manager

## 📖 Project Overview
This is a simple Task Manager application built with Flask as the backend and SQLite as the database.  
Users can add, update, and delete tasks.

## 🚀 Features
- ✅ Add new tasks  
- ✅ View all tasks  
- ✅ Update tasks  
- ✅ Delete tasks  

## 🛠️ Tech Stack
- **Backend**: Flask  
- **Database**: SQLite  
- **Frontend**: HTML, CSS
